All programs in this directory are intended for academic purposes.

Make sure the chromedriver is up to date and in the research directory.

Place a copy of the raw data in the raw folder before you parse!!

The parser (a.k.a. scraper) scripts pull the information from the raw data.

The interpreter scripts use the data to create the necessary plots.

The presentation scripts creates a power point with all the plots.

Gene Set Analysis Information:

Minimum number of IDs in the category: 5
Maximum number of IDs in the category: 2000
Significance Level: Top 25
Number of permutation: 1000

All data collected from Link Omics

DIRECTIONS: Run parse_all.py then interpret&present.r
